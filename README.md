
## Get Started

install ruby 2.2.2 (slightly older versions probably work)

```
cd open-data-maker
bundle
```

## Run the App

```
padrino start
```

go to: http://127.0.0.1:3000/


## Developer Notes

* [Padrino](http://www.padrinorb.com/) - Ruby Web framework
* [Liquid](http://liquidmarkup.org/) view templates
